Cool Palette Review

Six cool-hued color systems for the Susan Allen therapy site — each with token swatches, a mock application, and a one-line posture note. Pick one, or blend two, to lock before building the full mockup.

ClientSusan Allen, LMFT
Prepared byWellfull Collective
DateJuly 2026
Palettes6 cool variants

How this works

Six cool color systems, each mapped to the same six-token structure so they compare apples-to-apples. Swipe through to see how each palette feels applied to real site elements, then pick one — or blend two — to lock as the visual direction.

Step 01 Review each palette Six slides, each with token swatches and a mock application — nav, hero, card, and pull quote rendered in that palette's actual colors.
Step 02 Compare side-by-side A single grid puts all six accent bars and mini-swatches next to each other for quick visual scanning.
Step 03 Pick or blend Choose one variant, or tell us which elements to blend — e.g. "Variant A's accent with Variant D's background."
Step 04 Lock and build Once locked, the six tokens become the binding color contract. Typography (Cormorant Garamond + Atkinson Hyperlegible) stays fixed across all variants for fair comparison.
